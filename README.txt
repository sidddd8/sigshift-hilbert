Semestral project for Diagnostics and Testing

To run:
in terminal type "make"
after build, run with ./main [path_to_data]

[path_to_data] is an optional argument.
If not specified, data will be loaded from a filename as specified in SIG_CONFIG.h

shifts of signals are stored in .csv files in directory "shifts".
This directory also contains a python script to group all .csv to one.
